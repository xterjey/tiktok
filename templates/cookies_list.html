About Cookies Files

What are cookies files?

Cookies files contain your TikTok login information. The generator uses these files to authenticate with TikTok's servers and create streams on your behalf.

File format

The cookies file should be in JSON format and contain a list of cookies with name and value fields. 

[
  {
    "name": "sessionid",
    "value": "..."
  },
  {
    "name": "ttcid",
    "value": "..."
  }
]
